
# 🏆 Code Vipassana Season 14 ADK Visual Builder Agent

> **An Agentic AI Startup Advisor running with Google Cloud Agent Development Kit (ADK)**

This repository contains an autonomous AI startup advisor agent created using the low-code **ADK Visual Builder**. It designs ready-to-build, scalable architectures and monetization models powered by **Vertex AI** and **Google Search** tools.

---

## 🚀 Features
- **💡 Startup Idea Generation:** Automates brainstorming for highly profitable, real-world problems.
- **🏗️ Architectural Design Planning:** Proposes complete Google Cloud tech stacks (Firebase, Vertex AI, BigQuery).
- **💵 Monetization Frameworks:** Evaluates B2B and SaaS revenue models.
- **⚡ Prompt Engineering Optimized:** Utilizes continuous workflows designed for rapid responses.

---

- **Backend Model:** Gemini Extreme / Gemini 1.5 Flash
